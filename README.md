# Steps for building a docker'ed application

** Commands used **

- mvnw package
- find . -name \*\.jar -print
- docker build -t sea/my-stuff-backend:0.99 .
- docker image ls


## git commands

- mkdir quest-dockerfiles
- cd quest-dockerfiles/
- git init
- git remote add origin git@github.com:RS-F/quest-dockerfiles.git
